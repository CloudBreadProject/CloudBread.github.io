CloudBread 란?

PaaS Cloud

ClouBread는 PaaS 클라우드에 최적화된 엔진입니다. PaaS 에서 제공하는 여러 서버 리소스들을 효율적으로 활용합니다.

오픈소스 프로젝트

CloudBread는 MIT 라이센스스를 채택하고 있기 때문에 누구라고 쉽게 프로젝트 수정 및 재배포가 가능합니다.

모바일 게임 최적화 엔진

CloudBread는 모바일 게임을 빠르게 제작 할 수 있도록 도와줍니다. 제공되는 라이브러리들을 사용하면, 모바일 게임에 필요한 로직, 특화 기능 등을 구현 할 필요가 없어집니다.
CloudBread 의 특징
CloudBread 는 Microsoft Azure 를 바탕으로 만들어진 오픈소스의 프로젝트입니다.
CloudBread 의 다양한 프로젝트들이 진행되고 있으며, 이로 인해 다음과 같은 특징을 가질 수 있게 되었습니다.
게임 보안 인증 기능
CloudBread는 Microsoft의 Azure 를 백본으로 사용합니다. Azure에서 제공하는 보안 기능 을 사용하면 DB 에 대한 해킹 위험, 통신 프로토콜 패킷 해킹 등에 대한 위험을 줄일 수 있습니다. 또한 CloudBread 에서는 암호화 패킷을 통신 때문에 게임 보안에 대한 걱정을 줄일 수 있습니다.
다양한 게임 라이브러리 제공
CloudBread 는 약 100여가지의 게임 라이브러리를 제공합니다. 또한 모바일 게임에 특화된 SNS 기능, 채팅 기능 등을 제공하기 때문에, 필요한 라이브러리만 선택해서 사용함으로써, 서버 로직 개 발에 대한 수고를 줄일 수 있습니다.
Dev-Ops 지향 관리자웹 제공
CloudBread 는 게임 운영을 효율적으로 하기 위한 관리자 웹을 제공합니다. 이 웹에서 데이터베이 스 CRUD 기능을 사용 할 수 있으며, 쿠폰 관리, 푸시메시지 관리 등의 기능을 사용할 수 있습니다. 이를 통해 게임 개발에서 운영까지 한번에 해결 할 수 있습니다.
다양한 프로젝트 진행
CloudBread 는 오픈소스 프로젝트 입니다. 실제로 CloudBread 안에서 여러 프로젝트가 진행중 입니다. 관리자 웹 프로젝트, Unity SDK 등 CloudBread 에 필요한 프로젝트가 진행중이며, 앞 으로도 계속 추가해 나갈 계획입니다.
도큐멘테이션 및 예제 제공
CloudBread 는 개발자들을 위한 다큐멘테이션과 예제를 제공합니다. 저희 CloudBread 를 공부 하고, 사용하시는 개발자분들과 저희 프로젝트의 팀원들과의 상생을 위해, 다양한 방법으로 다큐멘테 이션과 예제를 제공하고 있습니다.
머신러닝을 사용한 사용자 분석
CloudBread 는 실제 게임을 운영하는 분들의 성공을 위해 사용자 분석 알고리즘을 개발하고 있습 니다. 머신러닝을 활용하여 알고리즘을 개발을 진행하고 있으며, 추후에 추가될 예정입니다.
