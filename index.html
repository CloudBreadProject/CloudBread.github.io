CloudBread 란?

PaaS Cloud

ClouBread는 PaaS 클라우드에 최적화된 엔진입니다. PaaS 에서 제공하는 여러 서버 리소스들을 효율적으로 활용합니다.

오픈소스 프로젝트

CloudBread는 MIT 라이센스스를 채택하고 있기 때문에 누구라고 쉽게 프로젝트 수정 및 재배포가 가능합니다.

모바일 게임 최적화 엔진

CloudBread는 모바일 게임을 빠르게 제작 할 수 있도록 도와줍니다. 제공되는 라이브러리들을 사용하면, 모바일 게임에 필요한 로직, 특화 기능 등을 구현 할 필요가 없어집니다.
CloudBread 의 특징
게임 보안 인증 기능

CloudBread는 Azure를 백본으로 사용하므로, Azure에서 제공하는 보안 기능을 사용가능.

다양한 라이브러리 제공

모바일 게임에 특화된 SNS, 채팅 공지 등 100여가지 게임 라이브러리를 제공.

Dev-Ops지향 관리자웹 제공

모바일 게임에 특화된 SNS, 채팅 공지 등 100여가지 게임 라이브러리를 제공.

다양한 프로젝트 진행

오픈소스 프로젝트로 CloudBread 에서 여러 프로젝트가 진행중.

도큐멘테이션 및 예제 제공

CloudBread 사용자분과 저희 팀원들의 상생을 위해, 다양한 문서 및 예제 제공.
